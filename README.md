本项目为自建的量化策略回测网站，目标是做出最好的量化策略，实现退休计划
项目启动命令：streamlit run app.py
app.py
├── 导入模块
├── 配置（API Key 等）
├── DeepSeek API 客户端
├── System Prompt（策略生成规范）
├── 代码提取与验证函数
├── Streamlit 界面
│   ├── 侧边栏（参数设置 + 策略选择）
│   │   ├── 内置策略（右侧/V型）
│   │   ├── 自定义策略（代码编辑器）
│   │   └── AI 生成策略（聊天界面）
│   ├── 主界面
│   │   ├── 运行回测
│   │   ├── 显示绩效指标
│   │   ├── 净值曲线
│   │   ├── K线图 + 成交量
│   │   ├── 最大回撤曲线
│   │   └── 交易明细
│   └── 会话状态管理
└── 回测引擎（run_backtest, calculate_metrics 等）
